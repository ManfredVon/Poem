待诗
===
我的诗词助手！
---

### 功能简介
* 用自己的手机创作自己的古典诗词
* 内置格律诗（五绝、七绝、五律、七律）格律和《钦定词谱》
* 可选择格律，边创作边看格律
* 可对诗词、草稿、格律进行搜索，快速查找
* 可分享诗词或格律到短信或QQ好友
